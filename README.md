# Portfolio Website

A personal portfolio website built using **HTML**, **CSS**, and **Bootstrap** to showcase my projects, skills, and experience in data science and software development.

## Features

- Responsive design using Bootstrap
- Projects section with descriptions and links
- Skills and experience overview
- Contact section

## Tech Stack

| Technology | Purpose |
|------------|---------|
| HTML5 | Page structure |
| CSS3 | Styling and layout |
| Bootstrap | Responsive design |

---

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/sohaib0075/Portfolio-website.git
   ```

2. Open `index.html` in your browser — no server needed!
